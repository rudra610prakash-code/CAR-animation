Car Animation Project

A visually engaging car driving animation built using HTML, CSS, and JavaScript.
The project demonstrates smooth CSS animations, moving background effects, rotating wheels, and background sound to create a realistic driving scene.

**Features**
Moving sky, trees, and road track using CSS animations
Rotating car wheels for realistic motion
Background engine sound using JavaScript audio
Pure front-end project (no libraries used)
Clean folder structure and beginner-friendly code

**Technologies Used**
HTML
CSS (Animations & Keyframes)
JavaScript (Audio control)

**Purpose of the Project**
This project was created to practice CSS animations, positioning, and JavaScript audio handling in a real visual scenario.
